이 프로젝트는 **플래시카드 학습법**과 **망각곡선(forgetting curve) 이론**을 적용하여  
영어 어휘와 한문 기초 한자를 효율적으로 학습할 수 있도록 돕는 웹 애플리케이션입니다.

## 🔗 논문 발표
본 프로젝트는 **한국컴퓨터교육학회 2024 하계학술대회**에서  
「한문 교육용 플래시 카드 시스템 개발 및 효과성 평가」라는 제목으로 발표되었습니다.
